Capstone Project Ideas

A structured vision for building a high-quality, curriculum-aligned AI system.

Module 1: Strategic Context Engineering & Hybrid RAG https://arxiv.org/html/2506.08938v1

Syllabus Focus: Fine-Tuning & RAG, Vector DBs, Hallucination Mitigation

System Goal: Build an intake router that maps multi-source inputs such as web content, transcripts, notes, and knowledge base documents into a structured schema before they reach the model.

Quality Mechanism: Implement a metadata-driven chunking strategy that tags every piece of information with Source Origin and Timestamp. This creates data lineage and helps the system understand where insights come from and how recent they are.

Module 2: Inference & Cache Optimization

Syllabus Focus: Pre- & Post-Training Optimization, KV Cache, Distillation, Quantization

System Goal: Organize the information matrix to maximize prompt caching efficiency while minimizing context window bloat.

Quality Mechanism: Build a two-stage reranking and compression pipeline. Stage 1 uses a lightweight cross-encoder such as BGE-Reranker to remove redundant or duplicated content. Stage 2 applies structural prefix shielding by ordering data from most static to most dynamic to improve prompt caching hit rates and reduce time-to-first-token.

Module 3: Alignment, Automated Grading, & Guardrails

Syllabus Focus: RL & Alignment, Evaluation & Benchmarking

System Goal: Build a programmatic validation gate that acts as a quality control filter before data reaches Dynamics 365.

Quality Mechanism: Implement an automated judge pipeline.

  • Step 1: Use NLI models or small language models to check Faithfulness and Contradiction Resolution. For example, if the web says “hiring” but the transcript says “layoffs,” the transcript should take precedence.
  • Step 2: Create a synthetic evaluation dataset of 50 complex lead scenarios with conflicting signals. Use metrics such as Context Recall and Answer Relevance through tools like Ragas or TruLens to prove the agent improves over a baseline web-search agent.
